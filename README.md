# Pokedex CLI

A command-line Pokédex written in Go. It's an interactive REPL that talks to the
public [PokeAPI](https://pokeapi.co/) so you can browse location areas, find
Pokémon, try to catch them, and inspect the ones you've caught.

## About this project

This is a **learning project** I built to practice Go. Along the way it touches a
bunch of core concepts:

- Building a REPL (read–eval–print loop)
- Making HTTP requests and unmarshalling JSON into structs
- Pointers and pointer semantics
- An in-memory cache for API responses
- Organizing code across multiple files and an `internal/` package

## Requirements

- Go 1.22 or newer

## Running it

Run directly:

```bash
go run .
```

Or build a binary and run it:

```bash
go build -o pokedexcli .
./pokedexcli
```

Either way you'll land at the `pokedex >` prompt.

## Usage

Type a command at the prompt. Some commands take an argument (a Pokémon name or a
location area).

```text
pokedex > help

Welcome to the Pokedex!
Usage:

help: Display a help message
map: Lists location areas
...

pokedex > map
canalave-city-area
eterna-city-area
pastoria-city-area
...

pokedex > explore pastoria-city-area
Exploring pastoria-city-area...
Found Pokemon:
- tentacool
- magikarp
...

pokedex > catch pidgey
Throwing a Pokeball at pidgey...
pidgey was caught!

pokedex > inspect pidgey
Name: pidgey
Height: 3
Weight: 18
Stats:
  -hp: 40
  -attack: 45
  -defense: 40
  -special-attack: 35
  -special-defense: 35
  -speed: 56
Types:
  - normal
  - flying

pokedex > pokedex
Your Pokedex:
- pidgey

pokedex > exit
Closing the Pokedex... Goodbye!
```

### Commands

| Command   | Argument      | Description                                                            |
| --------- | ------------- | --------------------------------------------------------------------- |
| `help`    | —             | Display the list of available commands                                |
| `map`     | —             | List the next page of location areas                                  |
| `mapb`    | —             | List the previous page of location areas                              |
| `explore` | `<area>`      | List the Pokémon found in a given location area                       |
| `catch`   | `<pokemon>`   | Attempt to catch a Pokémon (harder for higher base-experience Pokémon)|
| `inspect` | `<pokemon>`   | Show the name, height, weight, stats, and types of a caught Pokémon   |
| `pokedex` | —             | List all the Pokémon you've caught                                    |
| `exit`    | —             | Exit the Pokedex                                                       |

## Project structure

```text
pokedexcli/
├── main.go              # entry point: builds the client + config, starts the REPL
├── repl.go              # REPL loop and command registry
├── command_*.go         # one file per command (help, map, mapb, explore, catch, inspect, pokedex, exit)
├── repl_test.go         # tests
└── internal/
    └── pokeapi/         # PokeAPI client, response types, and the response cache
```
